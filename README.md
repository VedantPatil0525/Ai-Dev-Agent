# 🧠 AI Developer Assistant

An AI-powered developer assistant that helps automate common software development tasks such as code understanding, debugging, documentation generation, and unit test creation using Large Language Models (LLMs).

---

## 🚀 Features

* 🔍 **Code Explanation**
  Understand any code snippet in simple terms.
* 🐞 **Bug Detection & Fixing**
  Identify errors and get corrected code suggestions.
* 📄 **Documentation Generation** 
  Automatically generate structured documentation including function descriptions, parameters, and return values.
* 🧪 **Unit Test Generation**
  Generate test cases using `pytest` for better code reliability.
* ⚡ **Auto Mode (Optional)**
  AI decides the best action (explain/debug/document/test) based on the given code.

---

## 🛠️ Tech Stack

* Python 3.11
* Streamlit (UI)
* Ollama (Local LLM - LLaMA3)
* LangChain (Optional for advanced agent workflows)
